# BapCo TF2-Inspired Portfolio Redesign

This is a full HTML/CSS/JS redesign of BapCo's portfolio.

## What changed

- Rebuilt all pages with a TF2-inspired industrial/workshop aesthetic
- Added stronger portfolio wording for Roblox systems work
- Added responsive mobile navigation
- Added project cards, service cards, contact format, and achievement cards
- Removed messy inline styles and moved everything into `style.css`

## Pages

- `index.html` - homepage
- `whoami.html` - about page
- `projects.html` - project showcase
- `achievement2s.html` - achievements
- `portfolio.html` - contact / external portfolio link

## Deploying

Upload the full folder to GitHub Pages, Netlify, Vercel, or your existing host.
Keep `CNAME` if you are still using `bapco.online`.
